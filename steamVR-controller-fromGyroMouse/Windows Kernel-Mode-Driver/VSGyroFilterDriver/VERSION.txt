ВЕРСИЯ 1.0 - ПЕРВЫЙ ВЫПУСК
===========================

НОВЫЕ ФУНКЦИИ:
- Фильтрация шума малых движений мыши
- Сглаживание движения мыши (экспоненциальное сглаживание)
- Полная блокировка ввода мыши
- Управление параметрами фильтрации через IOCTL команды
- Поддержка Virtual HID Framework (VHF)

IOCTL КОМАНДЫ:
- IOCTL_GYRO_SET_BLOCK (0x800) - Полная блокировка ввода
- IOCTL_GYRO_SET_FILTER (0x801) - Включение/отключение фильтрации
- IOCTL_GYRO_SET_THRESHOLD (0x802) - Установка порога фильтрации
- IOCTL_GYRO_GET_INFO (0x803) - Получение информации об устройстве

УЛУЧШЕНИЯ:
- Обновлен SDK на Windows Driver Kit 10.0.19041.0
- Добавлена поддержка VHF через библиотеку vhfkm.lib

- Добавлены новые IOCTL команды для управления
- Создана полная документация

ИСПРАВЛЕНИЯ:

- VHF интеграция не полностью реализована
- Фильтрация работает только для стандартных MOUSE_INPUT_DATA структур

ПЛАНЫ НА БУДУЩЕЕ:
