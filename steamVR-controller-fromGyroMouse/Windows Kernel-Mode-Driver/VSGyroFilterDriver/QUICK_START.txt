КРАТКОЕ РУКОВОДСТВО ПО УСТАНОВКЕ И ИСПОЛЬЗОВАНИЮ
================================================

ШАГИ УСТАНОВКИ
==============

1. ПОДГОТОВКА ФАЙЛОВ
   - Скопировать следующие файлы в одну папку:
     * GyroFilterDriver.sys (из x64\Release\)
     * GyroFilterDriver.inf
     * gyrofilterdriver.cat (из x64\Release\GyroFilterDriver\)

2. ОТКЛЮЧЕНИЕ ПРОВЕРКИ ПОДПИСИ ДРАЙВЕРА (если необходимо)
   - Перезагрузить Windows в режиме отладки:
     bcdedit /set testsigning on
   - Перезагрузить компьютер

3. УСТАНОВКА ДРАЙВЕРА
   Способ 1 - Через Device Manager:
   - Открыть Device Manager (devmgmt.msc)
   - Найти устройство мыши
   - Правый клик -> Обновить драйвер
   - Выбрать "Обзор компьютера для поиска драйверов"
   - Указать папку с файлами драйвера
   - Нажать "Далее" и дождаться установки

   Способ 2 - Через командную строку (администратор):
   - Открыть Command Prompt как администратор
   - Перейти в папку с файлами драйвера
   - Выполнить: pnputil /add-driver GyroFilterDriver.inf /install

4. ПРОВЕРКА УСТАНОВКИ
   - Открыть Device Manager
   - Найти устройство мыши
   - Проверить, что установлен GyroFilterDriver

ИСПОЛЬЗОВАНИЕ
==============

1. КОМПИЛЯЦИЯ USER-MODE ПРИЛОЖЕНИЯ
   - Скомпилировать GyroMouseFilterControl.c:
     cl GyroMouseFilterControl.c /link user32.lib

2. УПРАВЛЕНИЕ ФИЛЬТРАЦИЕЙ

   Включить фильтрацию:
   GyroMouseFilterControl.exe enable-filter

   Отключить фильтрацию:
   GyroMouseFilterControl.exe disable-filter

   Установить порог фильтрации (10 пикселей):
   GyroMouseFilterControl.exe set-threshold 10

   Блокировать ввод мыши:
   GyroMouseFilterControl.exe block-input

   Разблокировать ввод мыши:
   GyroMouseFilterControl.exe unblock-input

   Получить информацию об устройстве:
   GyroMouseFilterControl.exe get-info

ОТЛАДКА
=======

1. ПРОСМОТР ОТЛАДОЧНЫХ СООБЩЕНИЙ
   - Скачать и запустить DebugView.exe
   - Запустить команды управления драйвером
   - Просмотреть сообщения в DebugView

2. ПРИМЕРЫ СООБЩЕНИЙ
   GyroMouseFilter: DriverEntry
   GyroMouseFilter: EvtDeviceAdd
   GyroMouseFilter: Device fully initialized
   GyroMouseFilter: Filtered small movement X=2 Y=3
   GyroMouseFilter: Filtered movement X=15 Y=12
   GyroMouseFilter: BLOCKED Delta X=0 Y=0

3. ПРОВЕРКА РАБОТЫ
   - Включить фильтрацию
   - Установить порог 10 пикселей
   - Медленно двигать мышь
   - Проверить, что малые движения игнорируются
   - Быстро двигать мышь
   - Проверить, что движение сглажено

УДАЛЕНИЕ ДРАЙВЕРА
=================

1. ЧЕРЕЗ DEVICE MANAGER
   - Открыть Device Manager
   - Найти устройство мыши
   - Правый клик -> Удалить устройство
   - Отметить "Удалить программное обеспечение драйвера"
   - Нажать "Удалить"

2. ЧЕРЕЗ КОМАНДНУЮ СТРОКУ (администратор)
   - Открыть Command Prompt как администратор
   - Выполнить: pnputil /delete-driver GyroFilterDriver.inf

3. ОТКЛЮЧЕНИЕ РЕЖИМА ОТЛАДКИ (если включали)
   - Открыть Command Prompt как администратор
   - Выполнить: bcdedit /set testsigning off
   - Перезагрузить компьютер

РЕШЕНИЕ ПРОБЛЕМ
===============

Проблема: Драйвер не устанавливается
Решение:
- Проверить, что все файлы (sys, inf, cat) находятся в одной папке
- Проверить, что включен режим отладки (testsigning on)
- Проверить, что устройство мыши не используется другим драйвером
- Перезагрузить компьютер

Проблема: Фильтрация не работает
Решение:
- Проверить, что драйвер установлен (Device Manager)
- Включить фильтрацию: GyroMouseFilterControl.exe enable-filter
- Проверить отладочные сообщения в DebugView
- Проверить, что порог фильтрации установлен правильно

Проблема: Мышь не работает после установки драйвера
Решение:
- Удалить драйвер через Device Manager
- Перезагрузить компьютер
- Переустановить драйвер
